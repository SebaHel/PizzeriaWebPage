# Pizzera - Italian Pizza Restaurant

Welcome to **Pizzera**, an Italian Pizza Restaurant website built using React. This project showcases a responsive and modern web interface for a pizzeria, allowing users to explore the menu, make reservations, and learn more about the restaurant.


## Features

- **Home Page**: A welcoming page with an overview of the restaurant's specialties.
- **Menu**: Browse through a variety of pizzas and Italian dishes.
- **Reservation Form**: Make a reservation by providing your details, preferred date, and time.
- **Contact Us**: Get in touch with the restaurant for any inquiries.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **React Router**: Used for managing navigation within the app.
- **CSS Modules**: For scoped and modular styling.
